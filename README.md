# Carapace Privacy Policy Site

This folder is ready to publish as a simple GitHub Pages site.

## Files

- `index.md` - public privacy policy page for App Store Connect

## Publish with GitHub Pages

1. Create a public repository and push these files.
2. In repository settings, enable **Pages**.
3. Set source to your default branch and root folder.
4. Use the published HTTPS URL in App Store Connect.

Example URL format:

- `https://<github-username>.github.io/<repo-name>/`
